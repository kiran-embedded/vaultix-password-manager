# рҹ”җ Vaultix

> **A modern offline-first password manager built with Flutter.**

Vaultix is designed around one simple principle:

> **Your passwords belong to you.**

No mandatory cloud accounts.

No central servers.

No subscriptions.

Only encrypted data that **you control**.

Everything is encrypted locally before storage. If you enable backup, only the encrypted vault is uploaded to **your own Google Drive**.

# рҹ”җ Security Architecture

Vaultix follows a **privacy-first** design where encryption happens **before** your data is written to storage or backed up.

No plaintext passwords are intentionally transmitted to external servers.
